Source code of the listening experiment:
"identification of material and aspect ratio by listening to rectangular plates"
as created within the Bachelor thesis of Michael Aurenhammer

Contains:
- Source code of the online experiment (2021, Michael Aurenhammer & Marian Weger), 
incl. patched jsPsych library "jspsych-repeat-audio.js"
- Matlab script for sound file generation (2021, Marian Weger)
- rendered sound files and images in "Dependencies.zip".

Link to experiment (as long as cognition.run hosts it):
https://29pfi4vsga.cognition.run/
